View Scope Server

An alternative explanation from MV*

View

This is your HTML, CSS, and your basic Javascript that Angular can help organize for you.

I categorize filters and directives as being a part of the view.

$Scope

This is the ViewModel, similar to an instance variable in a controller of rails.

def index @users = User.all.pluck('username') end

Server

Finally what is an app without a way for one computer to talk to the other?

I loosely categorize services and anything with $http as being part of the server, I also categorize fake server implementations such as loading some JSON in this as well.
